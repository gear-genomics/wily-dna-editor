The Wily DNA Editor is a everyday lab tool for handling sequences

The use of the tool should be intuitive and not require any help.
Paste a DNA sequence in the main field and try out to modify it using the buttons on top.
Other tabs offer digests and more...

There will be more content here once users start reporting problems.

 The Wily-DNA-Editor is still BETA! If you encounter any issues, find bugs, experience
 unintended or surprising behavior please drop us a note and give us the chance
 to improve Wily. We also enjoy positive feedback :-)
